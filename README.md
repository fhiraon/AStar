# AStar

A*(A-star)アルゴリズムをUnityで可視化してみた  

## 使い方
1. ブロックを置きたいマスをクリックする
2. Start ボタンを押す
3. もう一度行う場合はResetボタンを押す

![astar.gif](https://github.com/seinosuke/AStar/blob/master/images/astar.gif)

## マスの色の説明
* 赤 … スタート
* 青 … ゴール
* 黄 … 通過できないマス
* 橙 … 展開中のマス
* 緑 … Openリストに含まれるマス
* 灰 … Closedリストに含まれるマス
